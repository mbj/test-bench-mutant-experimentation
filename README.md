# Test Bench Mutant Experimentation

## Setup

Install gem dependencies:

``` sh
> ./install-gems.sh
```

Run tests with TestBench:

``` sh
> bundle exec bench
```

Run tests with RSpec:

``` sh
> bundle exec rspec
```

Run tests with Minitest:

``` sh
> ruby minitest/run.rb
```

IRB console:

``` sh
> ./console.sh
```
